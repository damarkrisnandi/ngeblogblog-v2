Hola~

Berikut ini merupakan cara simpel membuat postingan disini. Silahkan ikuti prosedurnya sampai selesai.

## Syarat
1. **NodeJS 18**. Syarat utama untuk bisa menjalankan Project ini adalah menggunakan **NodeJS 18+**, karena **NextJS** yg digunakan adalah versi 14, yang kompatibel minimal dengan **NodeJS 18** (artinya di atas versi 18 masih bisa digunakan)
2. Kalau pake **VSCode**, bisa dicoba Extension **MDX Preview** (tidak wajib tapi akan sangat membantu membuat postingan pada **VSCode**)
3. Udah itu aja. Yang wajib bahkan cuma nomor 1 aja.

## Installasi
install project ini dulu pake command berikut ini
```
npm install
```

## Compile Project di Local
Untuk running project di lokal, gunakan command berikut
```
npm run dev
```

## Create a New Post
Cara membuat postingan baru tidak perlu membuat file **.mdx** sendiri, isi sendiri formatnya. *Commandline* di bawah ini bisa digunakan untuk langsung membuatkan anda sebuah file baru.

```
npm run create-post <filename>
```

Contoh

```
npm run create-post new-post-1
```

maka akan terbentuk file bernama **20240701_2_new-post-1.mdx** pada folder ``/posts``. Format nama filenya sendiri akan menjadi ``yyyymmdd_<post ke berapa pada tgl tersebut>_<nama file yang diset di cli>.mdx``. Selanjutnya content filenya akan menjadi seperti berikut:

```
---
title: 'New Post'
description: 'Description of new post'
date: 'Mon Jul 01 2024 16:21:06 GMT+0700 (Indochina Time)'
tags: []
---
{/* customize your filename, title, desc, and all of this file properties */}
{/* fill your content here */}

```

Format di atas ini adalah format default jika kita membuat suatu postingan. Selanjutnya adalah kreativitas anda untuk mengisi postingan tersebut
